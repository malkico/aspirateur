# Démarrer

1- installer les dépendances :

```bash
npm i
```

2- démarrer le projet : 

```bash
npm start
```

3- tester le projet :
```bash
npm test
```


## À propos du projet
- Le code écrit est optimisé pour respecter les bonnes pratiques SOLID
- Une bibliothéque JEST est utilisée pour mettre en place des tests unitaires
